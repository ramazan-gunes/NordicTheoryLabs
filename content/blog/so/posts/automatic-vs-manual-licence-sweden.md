# Automatic mise manual imtixaanka Sweden? Waxa code 78 ka dhigan yahay

Marka aad qaadanayso laysanka B ee Sweden, waxaad dooran kartaa inaad ku tababarto automatic ama manual. Doorashadu waxay saamaysaa kharashka, kalsoonida iyo waxa aad kaxayn karto markaad baasto.

Haddii aad imtixaanka ku gasho automatic, Trafikverket wuxuu sheegayaa in shuruudda **78** lagu dari karo laysanka. Taas micnaheedu waa inaad kaxayn karto automatic oo keliya.

## Maxaa automatic loo doortaa?

Automatic wuxuu ka saaraa clutch iyo gear. Arday badan waxay markaas si fiican ugu fiirsadaan taraafikada, xawaaraha, booska gaariga, dadka lugaynaya iyo baaskiilayaasha.

Haddii aad noloshaada ku wadi doonto automatic, doorashadani way fududayn kartaa.

## Maxaa manual loo doortaa?

Haddii aad manual ku baasto, waxaad kaxayn kartaa manual iyo automatic labadaba. Taasi waxay faa'iido u leedahay shaqo, kiraysi ama baabuur duug ah.

Manual wuxuu qaadan karaa waqti dheer, laakiin wuxuu siinayaa xorriyad badan.

## Haddii aad rabto inaad code 78 ka saarto

Haddii laysankaagu leeyahay code 78 oo aad rabto manual, waxaad u baahan tahay imtixaan wadis cusub oo manual ah. Xeerarka hadda jira hubi ka hor ballansashada.

## Sida loo doorto

Dooro automatic haddii ujeeddadu tahay wadis maalinle ah oo automatic ah. Door manual haddii dabacsanaan mustaqbal muhiim kuu tahay.

Doorashadu waxay ku xiran tahay baahidaada, lacagta, stress-ka iyo gaariga aad ku tababaran karto.
