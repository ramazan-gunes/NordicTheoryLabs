# Permiso de aprendizaje y examen de vista en Suecia: primer paso para el permiso B

Antes de practicar para el permiso B en Suecia necesitas un permiso de aprendizaje válido, llamado **körkortstillstånd**. No es lo mismo que reservar una clase o empezar una app de teoría. Es la autorización oficial de Transportstyrelsen para practicar.

Sin este paso, el resto del proceso se complica. La autoescuela, el supervisor privado y la planificación de exámenes dependen de tener el permiso.

## Qué incluye la solicitud

La solicitud normalmente incluye una declaración de salud y un certificado de examen de vista. La declaración ayuda a evaluar si hay algo médico que afecte a la seguridad. El certificado de vista, **synintyg**, confirma que tu visión cumple los requisitos o que necesitas gafas o lentes.

El examen suele hacerse en una óptica o con otro proveedor autorizado. Pregunta siempre cómo se enviará el resultado antes de irte.

## Cuándo puedes empezar a practicar

Para la categoría B puedes empezar a practicar desde los **16 años** si tienes permiso válido. La edad para obtener el carnet es más alta, pero la práctica puede comenzar antes.

El permiso para practicar no significa estar listo para el examen. Solo abre la puerta. Aún necesitas formación, cursos de riesgo, teoría y los exámenes de Trafikverket.

## Si practicas de forma privada

La práctica privada tiene requisitos adicionales. Necesitas un supervisor aprobado, llamado **handledare**. Para coche, alumno y supervisor deben completar un curso introductorio de tres horas antes de practicar juntos.

El supervisor es responsable durante la práctica privada. Por eso conviene planificar con calma: rutas fáciles primero, objetivos claros y tráfico difícil solo cuando estés preparado.

## Errores comunes

El primer error es reservar clases o práctica privada antes de que el permiso esté aprobado. Empieza los trámites con tiempo.

El segundo error es tratar el examen de vista como un detalle. Si necesitas gafas o lentes, úsalas siempre durante la práctica.

El tercer error es confundir práctica privada y autoescuela. La práctica privada necesita supervisor aprobado. La autoescuela usa instructores autorizados.

## Lista simple

Solicita el körkortstillstånd, completa la declaración de salud, haz el examen de vista, espera la aprobación y después planifica la práctica. Si practicas en privado, confirma el supervisor y el curso introductorio.

Cuando este primer paso está claro, el proceso sueco del permiso de conducir se vuelve mucho más fácil de organizar.
