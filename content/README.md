# Content Structure

This folder is the source workspace for future localized site content.

## Blog

- `content/blog/_post-template.md` is the base markdown template for new posts.
- `content/blog/<lang>/posts/` is where localized article drafts should live.
- Public route placeholders are available at `blog/<lang>/`; English is canonical at `blog/` and `blog/en/` redirects there.

## Pages

- `content/pages/<lang>/` is reserved for localized copy for about, press, contact, changelog, and shared navigation text.
- Legal pages already have their own localized HTML under `privacy/` and `terms/`.

## Languages

Current language folders: `en`, `sv`, `tr`, `ar`, `fa`, `so`, `ku`, `pl`, `fi`, `bs`, `es`, `ru`.
