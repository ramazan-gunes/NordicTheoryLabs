Bidrag och stöd till körkort

Körkortet kostar 15 000–35 000 kr totalt. Det finns stöd att söka — här är vad som faktiskt finns och vem som kan söka det.

Spara på teoriprov med Körkort Hero

Arbetsförmedlingen — körkortsstöd vid jobbsammanhang

Arbetsförmedlingen kan bidra till körkortkostnaden om körkortet är direkt nödvändigt för ett specifikt jobb. Det är inte ett generellt bidrag — du behöver kunna visa att ett konkret jobbmöjlighet kräver körkort.

Vem kan söka?

  • Inskrivna på Arbetsförmedlingen som jobbeöskande
  • Deltagare i etableringsprogrammet
  • Personer som fått ett jobberbjudande som kräver körkort

Hur mycket?

Stödet varierar och är inte garanterat. I praktiken kan det täcka delar av kostnaden — ibland hela teoridelen och delar av körlektionerna. Beslut fattas individuellt av din handläggare.

Hur ansöker du?

  1. 1

    Kontakta din handläggare på Arbetsförmedlingen

    Berätta att du behöver körkort för ett jobb. Ta med jobbannonsen eller jobbkontakten som bevis.

  2. 2

    Handläggaren bedömer ärendet

    De tittar på ditt fall, dina chanser på arbetsmarknaden och om körkortet är rimligt att stödja.

  3. 3

    Beslut och betalning

    Om beviljat, betalar Arbetsförmedlingen direkt till trafikskolan — du betalar inte och får inte pengar kontant.

Kommunala stödprogram

Vissa kommuner har egna program för körkortstöd — framförallt för nyanlända, personer med försörjningsstöd eller långtidsarbetslösa. Utbudet varierar kraftigt mellan kommuner.

Kommuner med kända körkortsprogram

Malmö, Göteborg och Stockholm har historiskt haft riktade program. Kontakta socialförvaltningen eller arbetsmarknadsenheten i din kommun för aktuell information — program förändras varje år beroende på budget.

Spara pengar med rätt förberedelse

Det enklaste sättet att minska totalkostnaden är att klara provet på första försöket. Varje underkänt kunskapsprov kostar 450 kr extra. Varje underkänd uppkörning kostar 1 000–1 500 kr extra.

Körkort Hero är designad för att maximera din chans att klara teoriprov på första försöket — 1 287 frågor, offline, på 12 språk, med detaljerade förklaringar.

Vanliga frågor om bidrag till körkort

  • Kan Arbetsförmedlingen betala mitt körkort?

    Arbetsförmedlingen kan ge ekonomiskt stöd till körkort om det är nödvändigt för ett specifikt jobb du blivit erbjuden eller söker. Det är inte ett generellt bidrag — du behöver ett konkret jobbsamband. Kontakta din handläggare.

  • Ger CSN lån för körkort?

    Nej, CSN (studiemedel) täcker inte körkortskostnader. CSN gäller för godkända utbildningar på Komvux, folkhögskola, universitet m.m.

  • Kan kommunen hjälpa till med körkortet?

    Ja, vissa kommuner har specifika stödprogram för nyanlända eller långtidsarbetslösa. Det varierar kraftigt — kontakta socialförvaltningen eller arbetsmarknadsenheten i din kommun.

  • Vad är etableringsprogrammet och hur påverkar det körkortet?

    Om du deltar i Arbetsförmedlingens etableringsprogram kan din handläggare vid behov bevilja stöd för körkort om det bedöms öka dina chanser till jobb. Det är inte garanterat, men vanligare än många tror.

  • Kan jag ta körkort via SFI eller komvux?

    Körkortet i sig ges inte via SFI eller Komvux, men en del folkhögskolor erbjuder kombinerade utbildningar med körkortsförberedelse för nyanlända. Fråga din SFI-lärare eller Komvux om lokala möjligheter.
